### 介绍下BFC及其应用

- `BFC` 翻译过来指的是 `块级格式上下文`   
- 页面上一个 `隔离的渲染区域` 区域内的子元素的布局不会对外部元素造成影响     
- 触发BFC的元素
  1. `浮动元素`
  2. `float 值不为 none 的元素（浮动元素）`
  3. `绝对定位元素`
  4. `overflow 不为 visible 元素`
  5. `display 为 flex inline-block table-cell 的元素`
- 解决的问题
  1. 防止垂直边距重合
  2. 防止浮动元素造成父类元素的高度坍塌
  3. 防止浮动元素导致覆盖兄弟节点
